# Kesariya Balam

Illustrated Rajasthan travel guide web app — cities, places, food, and restaurants (by vibe), built with **Vite**, **React**, **TypeScript**, and **React Router**.

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — ESLint
- `npm run format` — Prettier

## Structure

Source lives under `src/` with `app/` (router), `components/`, `pages/`, `data/` (mock content), `styles/` (design tokens), and `assets/illustrations/` (SVG illustration icons).
